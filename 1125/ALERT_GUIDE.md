# 🔔 커스텀 알림 사용 가이드

## ✨ 새로운 기능

각 콘서트마다 원하는 날짜와 시간에 알림을 받을 수 있습니다!

---

## 📱 알림 설정 방법

### 1단계: 알림 토글 켜기
- 메인 화면에서 원하는 콘서트 카드의 **알림 토글** 클릭
- 알림 설정 모달이 자동으로 열립니다

### 2단계: 기본 알림 확인
- 알림을 켜면 **자동으로 공연 하루 전 알림**이 추가됩니다
- 설정된 알림 목록에서 확인 가능

### 3단계: 추가 알림 설정 (선택)
- 날짜와 시간 선택 (datetime-local 입력창)
- **"알림 추가"** 버튼 클릭
- 여러 개의 알림을 추가할 수 있습니다!

### 알림 끄기
- 알림 토글을 끄면 해당 콘서트의 **모든 알림이 삭제**됩니다

---

## 🎯 사용 예시

### 기본 알림 (자동)
- **콘서트**: 임영웅 IM HERO TOUR 2025 (2025년 11월 21일 17:00)
- **기본 알림**: 2025년 11월 20일 17:00 (하루 전)

### 추가 알림 설정
- **1차 알림**: 2025-11-14 20:00 (1주일 전)
- **2차 알림**: 2025-11-21 14:00 (당일 3시간 전)
- **3차 알림**: 자동 설정된 하루 전 알림

---

## 🔧 알림 관리

### 알림 수정하기
1. 콘서트 카드의 알림 토글 클릭 (모달 열기)
2. 기존 알림 삭제
3. 새로운 시간으로 알림 추가

### 모든 알림 삭제하기
- 알림 토글을 끄면 해당 콘서트의 모든 알림이 삭제됩니다

### 알림 다시 켜기
- 알림 토글을 다시 켜면 기본 알림(하루 전)이 자동으로 추가됩니다

---

## ⚠️ 중요 사항

### 1. 브라우저 알림 권한 필요
처음 사이트 접속 시 알림 권한을 허용해주세요!
- Chrome: 주소창 왼쪽 자물쇠 아이콘 → 알림 허용
- Firefox: 주소창 왼쪽 아이콘 → 알림 허용

### 2. 브라우저/탭이 열려있어야 함
- 현재 버전은 브라우저가 열려있을 때만 알림이 작동합니다
- 백그라운드 알림을 원하시면 **PWA로 설치**하세요!

### 3. 알림 체크 주기
- 앱이 30초마다 알림 시간을 체크합니다
- 설정한 시간 ±1분 이내에 알림이 발송됩니다

### 4. 과거 시간 설정 불가
- 현재 시간보다 이후의 시간만 설정 가능합니다
- 과거 시간 선택 시 경고 메시지가 표시됩니다

### 5. 기본 알림 자동 설정
- 알림을 켜면 공연 하루 전 알림이 자동으로 추가됩니다
- 공연이 하루 이내인 경우 기본 알림이 추가되지 않습니다

## 💡 활용 팁

### 📌 팁 1: 기본 알림 활용
기본 알림(하루 전)으로 충분하다면
- 토글만 켜고 모달 닫기
- 자동으로 설정된 알림 활용

### 📌 팁 2: 예매 알림
예매 시작 10-15분 전 알림 추가
- 예매 사이트 접속 여유 시간 확보
- 티켓팅 준비 완료

### 📌 팁 3: 공연 준비 알림
공연 3-4시간 전 알림으로 준비 시간 확보
- 교통편 확인
- 옷차림 체크
- 티켓/신분증 확인

### 📌 팁 4: 중복 알림 방지
- 기본 알림(하루 전)으로 충분한지 확인
- 필요한 알림만 추가

## 🎵 PWA 설치로 더 나은 경험

### PWA(Progressive Web App) 설치하면:
✅ 백그라운드 알림 가능성 향상
✅ 앱처럼 사용 가능
✅ 홈 화면에 아이콘 추가
✅ 오프라인에서도 작동

### 설치 방법:
1. 브라우저에서 사이트 접속
2. 주소창 오른쪽의 "설치" 아이콘 클릭
3. 홈 화면에 추가

## ❓ 자주 묻는 질문

### Q: 알림이 오지 않아요
**A:** 다음을 확인해주세요:
1. 브라우저 알림 권한 허용 확인
2. 브라우저/탭이 열려있는지 확인
3. 알림 시간이 미래 시간인지 확인
4. 시스템 알림 차단 설정 확인

### Q: 기본 알림만 사용하고 싶어요
**A:** 알림 토글만 켜고 모달을 닫으면 됩니다. 자동으로 하루 전 알림이 설정됩니다.

### Q: 알림을 끄면 어떻게 되나요?
**A:** 해당 콘서트의 모든 알림이 삭제됩니다. 다시 켜면 기본 알림(하루 전)이 재설정됩니다.

### Q: 여러 개의 알림을 설정할 수 있나요?
**A:** 네! 각 콘서트마다 원하는 만큼 알림을 추가할 수 있습니다.

### Q: 알림 시간을 수정하려면?
**A:** 기존 알림을 삭제하고 새로운 시간으로 다시 추가해주세요.

### Q: 모바일에서도 작동하나요?
**A:** 네! 모바일 브라우저에서도 동일하게 작동합니다. PWA로 설치하면 더 좋습니다.

## 🆘 문제 해결

### 알림이 작동하지 않을 때
1. **브라우저 새로고침** (F5)
2. **캐시 삭제** (Ctrl + Shift + Delete)
3. **알림 권한 재설정**
4. **브라우저 업데이트**

### 알림 모달이 안 열릴 때
- 알림 토글을 다시 켜보세요
- 브라우저를 새로고침하세요
