🔮 Mystic Flow: 당신의 손안에 펼쳐지는 AI 타로

1. 서비스 소개 (Service Overview)

"고민이 있으신가요? 별들에게 물어보세요."

**Mystic Flow(미스틱 플로우)**는 현대인의 불안과 고민을 위로하기 위해 탄생한 생성형 AI 기반의 타로 리딩 웹 애플리케이션입니다.
전통적인 타로 카드의 신비로운 지혜와 최신 DeepSeek LLM(거대언어모델) 기술을 결합하여, 언제 어디서나 나만의 전담 타로 마스터와 대화하는 듯한 깊이 있는 상담 경험을 제공합니다.

2. 기획 의도 (Background)

문제 제기: 기존의 타로 앱들은 정해진 데이터베이스에서 문장을 무작위로 출력하여, 내 구체적인 상황(취업, 짝사랑, 인간관계 등)에 맞는 답변을 주지 못했습니다. 오프라인 점집은 시간과 비용의 부담이 컸습니다.

해결책: 우리는 생성형 AI의 '문맥 이해 능력'에 주목했습니다. 사용자의 구체적인 고민을 AI가 이해하고, 뽑힌 카드의 상징과 연결하여 세상에 하나뿐인 해석을 생성합니다.

3. 핵심 경쟁력 (Key Features)

✨ 1. 초개인화된 AI 리딩 (Hyper-Personalized AI)

단순한 랜덤 운세가 아닙니다.

"내가 지금 썸 타는 사람과 연락이 끊겼는데..." 라는 사용자의 구체적인 고민 텍스트를 AI가 분석하고, 선택된 카드의 의미를 재해석하여 마치 친구가 조언해주듯 따뜻하고 구체적인 답변을 제공합니다.

📱 2. 설치 없는 앱 경험 (PWA Technology)

앱스토어에서 검색하고, 다운로드하고, 회원가입하는 번거로움이 없습니다.

접속 즉시 실행: 링크(URL)만 누르면 바로 앱이 실행됩니다.

홈 화면 추가: 스마트폰 바탕화면에 아이콘을 추가하여 네이티브 앱처럼 사용할 수 있습니다.

오프라인 지원: 인터넷이 불안정해도 기본 화면이 유지됩니다.

🎨 3. 몰입감을 주는 UI/UX

Dark Mystical Theme: 우주와 심해를 연상시키는 딥 네이비 컬러와 금빛 포인트로 신비로운 분위기를 연출했습니다.

Interactive Animation: 실제 카드를 펼치고, 뒤집고, 섞는 듯한 부드러운 애니메이션으로 타로 리딩의 손맛을 살렸습니다.

4. 사용 방법 (User Guide)

접속하기: Mystic Flow 웹사이트에 접속합니다.

고민 입력: 이름과 생년월일(선택), 그리고 현재 마음속의 고민을 구체적으로 적습니다.

카드 집중: 마음을 가다듬고 뒷면으로 펼쳐진 78장의 카드 중, 끌리는 3장을 선택합니다.

AI 해석: AI가 당신의 고민과 카드를 연결하는 동안 잠시 기다립니다.

결과 확인: [과거-현재-미래]의 흐름과 구체적인 조언을 확인하고, 결과를 캡처하여 소장합니다.

5. 팀 소개 (Team GenCoder)

우리는 'FLOW' 협업 툴을 통해 기획부터 개발까지 물 흐르듯 유연하게 소통하며 이 프로젝트를 완성했습니다.

기획 & 디자인: 사용자 경험을 설계하고 신비로운 비주얼을 구현.

프론트엔드: React와 PWA 기술로 끊김 없는 사용성 구축.

AI 엔지니어링: DeepSeek API를 튜닝하여 점술가 페르소나 주입.

Mystic Flow는 단순한 앱이 아닙니다. 당신의 마음을 비추는 디지털 거울입니다.
